# 知恵ネキの処世術

「上手く生きる知恵」を扱う Shorts チャンネル用の作業領域です。

## 主な内容

- `ideas/` — 企画、テーマバックログ、現在の表現方針
- `prompts/` — 台本・メタデータ生成用プロンプト
- `metadata/generated/` — 投稿用メタデータ、動画仕様 JSON
- `assets/generated/` — 生成画像、音声、manifest などの動画素材
- `renders/` — 書き出し済み mp4 と確認用 contact sheet
- `automation/` — アップロード、予約投稿、検証ログ
- `branding/` — チャンネルアイコンやブランド方針
- `scripts/` — 知恵ネキ向けの動画生成・アップロード補助スクリプト

## 投稿・制作方針

既存の `release_014_reply_trust_grouped_fullcaptions` 系ワークフローを基準にします。
新規動画では各動画・各シーンの画像を新規生成し、YouTube Studio で公式 BGM を低音量追加します。
