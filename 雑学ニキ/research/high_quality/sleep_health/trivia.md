# 睡眠・健康（断定注意）

## 01. 睡眠は「免疫の記憶」にも関わる可能性がある
- 一言フック: 寝ることは、脳だけでなく免疫にも関係する。
- ショート台本案: 睡眠は記憶を整理する時間と言われるが、免疫にも関係する可能性がある。小規模研究では、ワクチン後に睡眠を取った群で免疫記憶の反応が高まったと報告された。
- 映像案: 寝ている人、盾、メモリーカード。
- 評価: 意外性4 / 動画化4 / 信頼度3
- 注意: 医療助言にしない。「可能性」「研究では」に留める。
- source_url: https://pubmed.ncbi.nlm.nih.gov/21632713/

## 02. 停電時の冷蔵庫は、開けないだけで安全時間が伸びる
- 一言フック: 非常時の冷蔵庫対策は、まず開けないこと。
- ショート台本案: USDAによると、停電時の冷蔵庫はドアを閉めたままなら約4時間冷たさを保てる。中身を確認したくても、開けるほど冷気は逃げる。
- 映像案: 停電、冷蔵庫、開ける手を止める。
- 評価: 実用性5 / 動画化4 / 信頼度5
- source_url: https://www.fsis.usda.gov/food-safety/safe-food-handling-and-preparation/food-safety-basics/keeping-food-safe-during

## 03. 匂いが昔の記憶を連れてくるのは脳の構造と関係する
- 一言フック: 匂いで一瞬で昔に戻る現象には、脳の理由がある。
- ショート台本案: 匂いの処理は、感情や記憶に関わる脳領域と結びつきやすい。だから音楽や写真よりも、匂いが急に強い記憶を呼び起こすことがある。
- 映像案: 香り、記憶の扉、昔の教室。
- 評価: 意外性4 / 動画化5 / 信頼度4
- source_url: https://pmc.ncbi.nlm.nih.gov/articles/PMC3794443/

## 04. 動物の異常行動だけで地震予知はできない
- 一言フック: ペットの様子だけで地震を当てるのは科学的に難しい。
- ショート台本案: 地震前に動物が変な行動をしたという話はある。でもUSGSは、特定の行動と地震発生の再現性ある関係は確認されていないとしている。動物は敏感でも、予言者ではない。
- 映像案: 犬猫、地震計、再現性にバツ。
- 評価: 誤情報対策5 / 動画化4 / 信頼度5
- source_url: https://www.usgs.gov/programs/earthquake-hazards/earthquake-facts-earthquake-fantasy

## 05. 猫のゴロゴロ音は、安心だけでなく不調サインの場合もある
- 一言フック: 「ゴロゴロ＝幸せ」と決めつけない。
- ショート台本案: 猫は嬉しい時にもゴロゴロ鳴く。でも不安や痛みがある時にも鳴く場合がある。鳴き声だけで判断せず、食欲や動きなど全体を見るのが大事。
- 映像案: 幸せ猫、元気のない猫、観察リスト。
- 評価: 実用性4 / 動画化5 / 信頼度4
- source_url: https://www.loc.gov/everyday-mysteries/zoology/item/how-do-cats-communicate-with-each-other/
