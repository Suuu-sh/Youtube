# 人体・脳・心理

## 01. 匂いは記憶と感情を強く呼び起こしやすい
- 一言フック: 懐かしい匂いで昔を思い出すのは、気のせいではない。
- ショート台本案: 匂いの情報は、感情に関わる扁桃体や記憶に関わる海馬と結びつきやすい。だから香水、給食、雨の匂いで昔の場面が急に戻ってくることがある。
- 映像案: 匂いの煙、脳、思い出写真。
- 評価: 意外性4 / 動画化5 / 信頼度4
- source_url: https://pmc.ncbi.nlm.nih.gov/articles/PMC3794443/

## 02. 体内のセロトニンの多くは脳ではなく腸で作られる
- 一言フック: 幸せホルモンと呼ばれるセロトニンの主な生産地は腸。
- ショート台本案: セロトニンは脳の話として語られがち。でも研究レビューでは、体内セロトニンの大部分は消化管で作られるとされる。ただし腸のセロトニンがそのまま脳に入るわけではない点に注意。
- 映像案: 脳と思わせて腸にズーム。
- 評価: 意外性5 / 動画化4 / 信頼度4
- source_url: https://pmc.ncbi.nlm.nih.gov/articles/PMC9441471/

## 03. 運動は脳の記憶回路にも影響する
- 一言フック: 運動で鍛えられるのは筋肉だけじゃない。
- ショート台本案: マウス研究では、回し車で走ることで海馬の新しい神経細胞が増え、回路も変化することが示されている。運動は体だけでなく、記憶に関わる脳の仕組みにも影響する。
- 映像案: ランニング、脳の海馬、電気信号。
- 評価: 意外性4 / 動画化4 / 信頼度4
- source_url: https://pmc.ncbi.nlm.nih.gov/articles/PMC6422697/

## 04. 眠った後のワクチン接種反応は強くなる可能性がある
- 一言フック: 睡眠は記憶だけでなく、免疫の記憶にも関わる。
- ショート台本案: 小規模研究では、ワクチン接種後に睡眠を取った群で免疫記憶に関わる反応が高まったと報告されている。睡眠は脳を休めるだけでなく、体の学習時間でもある。
- 映像案: 注射、睡眠、盾マーク。
- 評価: 意外性4 / 動画化4 / 信頼度3
- 注意: 医療助言ではなく研究紹介として扱う。
- source_url: https://pubmed.ncbi.nlm.nih.gov/21632713/

## 05. 猫が嬉しい時だけゴロゴロ鳴るとは限らない
- 一言フック: 猫のゴロゴロ音は、幸福だけのサインではない。
- ショート台本案: 猫は安心している時にもゴロゴロ鳴く。でもLibrary of Congressの解説では、不安や痛みがある時にも鳴く場合があるとされる。かわいい音でも、状況を見るのが大事。
- 映像案: リラックス猫、病院猫、注意マーク。
- 評価: 意外性4 / 動画化5 / 信頼度4
- source_url: https://www.loc.gov/everyday-mysteries/zoology/item/how-do-cats-communicate-with-each-other/
