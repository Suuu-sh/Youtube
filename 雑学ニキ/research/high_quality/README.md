# 高品質雑学ストック（試作）

雑学ニキ用に、動画化しやすい雑学だけをカテゴリ別に保存する場所。
全文転載ではなく、公開情報をもとにした「要点・台本化メモ・検証URL」を保存する。

## 採用基準

- 1秒で引っかかる意外性がある
- 15〜30秒のShortsにしやすい
- 画像・いらすとや・簡単な図解で見せやすい
- 信頼できる機関・研究・公的資料で裏取りしやすい
- 医療・金融・法律のような高リスク情報は断定を避ける

## 使い方

各カテゴリの `trivia.md` から「一言フック」と「ショート台本案」を台本化する。
`source_url` は表に出さなくてもよいが、誤情報チェック用に残す。
