# 歴史・文化

## 01. 自由の女神は、200箱以上に分けられてニューヨークへ来た
- 一言フック: 自由の女神は、完成品ではなく巨大なパーツとして届いた。
- ショート台本案: 1885年、フランス船イゼール号はニューヨーク港に到着した。積んでいたのは、自由の女神になる銅と鉄の巨大パーツ。スミソニアンによると、その数は200箱以上。あの像は、海を渡って組み立てられた巨大キットだった。
- 映像案: 木箱、船、組み上がる像。
- 評価: 意外性5 / 動画化5 / 信頼度5
- source_url: https://americanhistory.si.edu/explore/stories/looking-liberty-nation-we-build-together

## 02. 自由の女神の中身は、銅だけでなく鉄の骨組みも重要
- 一言フック: 自由の女神は、銅の皮と鉄の骨で立っている。
- ショート台本案: 自由の女神は銅像として知られるが、巨大な銅板を支える骨組みには鉄が使われた。スミソニアンの資料では、銅板とそれを支える鉄構造が材料選択の重要点として扱われている。
- 映像案: 外側の銅、内側の骨組み透視図。
- 評価: 意外性4 / 動画化5 / 信頼度4
- source_url: https://mci.si.edu/node/1229290

## 03. アメリカのサツマイモが「ヤム」と呼ばれる背景には歴史がある
- 一言フック: 食べ物の呼び名には、移民と農業の歴史が残る。
- ショート台本案: アメリカでは柔らかいタイプのサツマイモが「ヤム」と呼ばれることがある。Library of Congressは、アフリカ由来のヤムに似ていたため、区別のためにそう呼ばれた背景を紹介している。
- 映像案: アフリカのヤム、アメリカのサツマイモ、ラベル。
- 評価: 意外性4 / 動画化4 / 信頼度5
- source_url: https://www.loc.gov/everyday-mysteries/item/what-is-the-difference-between-sweet-potatoes-and-yams/

## 04. 冷凍食品は、発明というより「自然現象の観察」から広がった
- 一言フック: 便利な冷凍食品の始まりは、極寒の魚の観察だった。
- ショート台本案: Birdseyeは寒冷地で魚が一気に凍る様子を観察し、急速冷凍なら味や食感を保ちやすいと気づいた。近代食品産業の大きな発明は、自然の観察から始まった。
- 映像案: 氷点下の湖、魚、工場ライン。
- 評価: 意外性4 / 動画化5 / 信頼度5
- source_url: https://www.loc.gov/everyday-mysteries/technology/item/who-invented-frozen-food/

## 05. グランドキャニオンは「若い地形」だが、岩は超古い
- 一言フック: 景色としてのグランドキャニオンと、岩の年齢は桁が違う。
- ショート台本案: グランドキャニオンの地形は主に数百万年で削られた比較的若いもの。でも露出している岩は15億年以上の歴史を含む。見えている景色は若く、材料は古い。
- 映像案: 侵食する川、古い岩の年代表。
- 評価: 意外性5 / 動画化4 / 信頼度5
- source_url: https://www.nps.gov/places/000/grand-canyon-s-rocks-are-incredibly-old.htm
