## Description

食べ物・飲み物の成分と性質を深めに知るLv4雑学を短く紹介。

【詳細・補足】
1. 紫キャベツの色は 酸とアルカリで 変わる
紫キャベツの色素アントシアニンは、周りの酸性・アルカリ性で色が変わりやすい。理科実験のpH指示薬としてよく使われる。

2. こんにゃくの弾力は 食物繊維が 水を抱える
こんにゃく芋のグルコマンナンは水を多く抱え、アルカリ条件で網目状に固まりやすい。低カロリーで腹持ちが出やすい理由にもつながる。

3. クランベリーには 細菌の付着を 邪魔する成分がある
プロアントシアニジンという成分が、細菌が尿路の表面につく動きを妨げる可能性がある。効果は製品や摂り方で変わるため万能ではない。

4. きゅうりの苦味は 防御用の成分が 関わる
ウリ科の植物にはククルビタシンという苦味成分が関わることがある。品種改良で強い苦味は抑えられるが、栽培環境で感じ方が変わることもある。

5. ピーナッツは 木の実ではなく 豆の仲間
名前にナッツとつくが、植物分類ではマメ科。花のあと子房が土にもぐり、地中でさやを作るところも特徴的。
