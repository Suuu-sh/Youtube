## Description

9割が知らない面白い雑学。からだとクセの、つい話したくなる日常ネタを5つまとめた。

【詳細・補足】
1. あくびは見るだけでつられて出ることがある
あくびは、見たり聞いたり考えたりするだけで起こることがある。脳が他人の動きに反応する現象の一つで、眠気だけが理由とは限らない。
出典: https://www.sleepfoundation.org/sleep-faqs/why-do-you-yawn

2. 耳あかはあごの動きで少しずつ外へ運ばれる
耳あかは外耳道を守るための自然な分泌物。耳の中の皮膚の移動やあごの動きで外へ出るため、奥まで押し込む掃除は詰まりの原因になることがある。
出典: https://www.health.harvard.edu/a_to_z/wax-blockage-of-the-ear-canal-a-to-z

3. ため息は気分だけでなく肺をふくらませ直すリセット呼吸
ため息は落ち込みのサインだけでなく、体が自然に行う深い呼吸でもある。研究では、肺の小さな袋をふくらませ直す働きと関係すると説明されている。
出典: https://www.nature.com/articles/nature16964

4. 貧乏ゆすりは座ったまま消費を少し増やす動き
脚を小さく動かす行動は、完全にじっと座るよりエネルギー消費が増えやすい。もちろん運動の代わりにはならないが、座りっぱなしを中断する小さな動きとして研究されている。
出典: https://www.nature.com/articles/s41598-025-33921-8

5. 紙で指を切ると痛いのは神経が多い場所を浅く切るから
指先は細かい感覚を拾うため、神経の密度が高い場所。紙の切り傷は小さくても浅い層を鋭く傷つけるため、見た目以上に痛みを感じやすい。
出典: https://health.osu.edu/health/general-health/why-do-papercuts-hurt
