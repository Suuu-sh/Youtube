## Description

9割が知らない面白い雑学。家・学校・職場でふと気になる身近な謎を5つまとめた。

【詳細・補足】
1. 風呂で歌がうまく聞こえるのは音が反響して厚くなるから
浴室は硬い壁が多く、声が吸収されにくい空間。反響が声に重なって音量や厚みが増し、普段より気持ちよく聞こえやすい。
出典: https://science.howstuffworks.com/life/biology-fields/sing-in-the-shower.htm

2. 靴ひもがほどけるのは着地と足振りが結び目をゆるめる
歩くたびに足が地面へ当たり、結び目には衝撃が入る。同時に足の振りでひもの端が引っ張られ、ほどける方向へ力が重なる。
出典: https://news.berkeley.edu/2017/04/11/shoe-string-theory-science-shows-why-shoelaces-come-untied/

3. 冷たい物で頭キーンは口の奥の冷えを頭の痛みと読む
冷たい食べ物が上あごや喉の奥を急に冷やすと、血管や神経が反応する。脳はその信号を頭の痛みとして受け取り、短い頭痛として感じることがある。
出典: https://www.health.harvard.edu/pain/what-causes-ice-cream-headache

4. 水で指がシワになるのは神経が血管を細くする反応
長く水に触れた指のシワは、ただ水を吸っただけでは説明しにくい。自律神経が血管を縮める反応が関わり、濡れた物をつかみやすくする働きも考えられている。
出典: https://www.medicalnewstoday.com/articles/322705.php

5. 香りで昔を思い出すのは記憶と感情の近くへ直接届くから
においの情報は、感情や記憶に関わる脳の領域と近い経路で処理される。だから特定の香りが、場所や人の記憶を一気に呼び戻すことがある。
出典: https://health.clevelandclinic.org/smells-and-memory
